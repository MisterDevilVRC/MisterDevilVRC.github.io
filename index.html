YouTube Sync for VRChat

Eliminating the browser-tab workflow for Quest and PC world-makers.

The Goal

Allow VRChat friends to watch the latest content from curated YouTube channels without ever leaving the world. By bypassing traditional search, we ensure the list is always fresh and updated via a backend automation script.

Workflow

1. Python: Fetches 50 videos per channel & sorts by date.
2. Gist API: Compiles and posts raw data links.
3. Udon: Fetches data in-game and renders interactive buttons.

Build Status

Python Logic 100%
Udon Integration 10%

Project Vision

"Quest friends complain about link copying—it's frustrating. This project is about making shared viewing as easy as pressing a single button inside the world."
