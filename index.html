Project Development

Optimized content synchronization for VRChat worlds.

The Goal

Allow VRChat friends to watch the latest content from curated YouTube channels without ever leaving the world. By bypassing traditional search, we ensure the list is always fresh and updated via a backend automation script.

Workflow

• Python: Automation logic for API data fetching.
• Gist API: Secure cloud hosting for video metadata.
• Udon: Real-time in-world data parsing and UI generation.
