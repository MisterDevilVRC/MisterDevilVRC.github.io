Project Under Development

The Goal

An asset for VRChat world makers to easily integrate YouTube playlists. This project eliminates the need for players to alt-tab to browsers or copy-paste links manually. Stay updated with the latest content from multiple channels without the clutter of years-old search results.

How It Works

  • Configuration: Provide your YouTube API Key, GitHub Key, and Target Channel IDs.
  • Data Pull: A Python script fetches the 50 most recent videos per channel.
  • Sorting: Results are mixed and chronologically sorted by upload date.
  • Sync: The list is pushed to a GitHub Gist (or Pastes.io) via API.
  • Udon Prefab: Place the prefab in your Unity world and link it to the raw Gist data.

Project Progress

Python Backend (Data Processing) 100%
Udon Prefab / Unity UI 10%

Current Status: Text is readable in-world, but clickable interaction for links is still in development.

The Motivation

"I watch a lot of funny videos in VRChat with PC and Quest friends. It’s annoying to tab out, and Quest friends can't easily copy links. I wanted a way to just press 'Play' on the latest content without leaving the game."
